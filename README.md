# NTechFix Scripts & Automation Repository

Welcome to the public **NTechFix** repository.

This repository contains a growing collection of PowerShell, Python, and JavaScript projects focused on IT administration, automation, reporting, Microsoft 365, SharePoint, asset management, documentation, and workflow improvement.

Many of these tools were originally created to solve real-world operational challenges, automate repetitive tasks, improve reporting, and streamline administration. As projects mature, they are cleaned up, documented, and published here for others to learn from or adapt to their own environments.

# 📚 Documentation Philosophy

One of the primary goals of this repository is ensuring projects remain understandable months or years after they were originally written.

Whenever possible, projects include:

- README documentation
- Installation instructions
- Configuration guidance
- Usage examples
- Sample outputs
- Project background and purpose

The goal is simple:

> Future me should be able to understand exactly how a project works without reverse engineering my own scripts.
